# Running a port
## Terminal
Before running, you will need to make sure you have the commands `stty` and `dd` installed. Both of these programs are required for getting user input.

Open a terminal, go to the `lua/` directory inside the repo, then type `lua* terminal.lua`, and replace the asterisk with your Lua version.
